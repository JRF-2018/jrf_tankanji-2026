## このアーカイブに含まれるファイル

このアーカイブに含まれるファイルの漢字エンコードは Perl スクリプトやド
キュメントは UTF-8 なのですが、データの漢字エンコードが euc-jisx0213 と
いう特殊なものになっています。

前回(jrf_tankanji-20120505.zip)の公開に比べて Perl 自体は UTF-8 化した
のですが、中で扱っているのは euc-jisx0213 です。メインの生成物の
tankanji-2026.txt も euc-jisx0213 です。

メインの配布物は『風』互換ソフト用(例えば [quail-naggy.el](https://github.com/JRF-2018/quail-naggy))に作ったテキスト形式
の単漢字辞書になります。やや実験的ですが、単漢字変換の動態保存のために、
2026年になって再作成＆再公開することにしました。

  * tankanji-2026.txt

今回の辞書の特徴は、これまでの風辞書や2012年の辞書と違い、漢字の出てく
るキーボードの場所は同じだが、ページは読みごとに違うことを認めるように
した点です。風よりも多くの文字を扱うため、これまではどうしてもページの
後ろのほうに表示される文字が多かったのですが、それをこの変更で、前のほ
うに表示できるようにしました。邪道という批判は甘んじて受けます。


これを作るために使ったスクリプトと、ネットで手に入れた以外の私が中間的
に作った補助情報テキストを同梱しています。他の素材をネット等で手に入れ
る必要があるため、これらだけでスクリプトは動きません。他の素材はすでに
入手困難となっており、2026年現在、インターネットアーカイブ等を頼る必要
があります。

スクリプトは自分の作業で使うことしか考えてないので、作業は直接スクリプ
トをいじってモードを変える形のままであるなど、とても汚いです。(2012年
に作った時期が精神病での入院からのリハビリ期にあたるということもありま
す。) 2026年になって、use strict や use warnings も通るようにまず変更
し、かなりの機能追加を行いました。

配布物には以下が含まれます。

  * extract-tankanji.pl (スクリプト本体)
  * bushu-name.euc.txt
  * bushu-info.euc.txt
  * predef-tankanji.euc.txt
  * yomi-ignore.euc.txt
  * joyokanji_gakunen.euc.txt
  * SKK-JISYO.hoi
  * kana-table.euc.pl (スクリプト用のライブラリ)
  * dakuon-table.euc.pl (スクリプト用のライブラリ)

実は2012年の公開時は、スクリプトをそのまま実行しても tankanji.txt は生
成できないまま公開してしまいました。2005年ぐらいをメインに
tankanji.txt を作っていた実験をいろいろ忘れてしまっていたのです。

今回は、ちゃんと生成したそのままをお出ししています。

あと、人に読まれることが前提になってなかった作業履歴と TODO リストも同
人的配布に甘えて同梱しています。TODO リストは、むしろ、仕様のメモになっ
ていて、実装した部分も含まれています。

  * 00LOG.txt
  * 00TODO.txt

以下は識別用の URI が書かれたファイルです。書かれた URI からはダウンロー
ドできません。

  * 00URI.txt

そして、現在お読みのこのファイルになります。

  * README.md

## 配布条件

このアーカイブは同人的に配布されています。特殊な配布条件があります。
2012年当時は以下のような文言で配布していました。

《まさかの単漢字入力。日本語 IME『風』 - JRF の勝手に PR》  
http://jrf.cocolog-nifty.com/pr/2012/04/post-4.html  

> スクリプトの最初を読めばわかるように、ネットでフリーに手に入るものを
> 中心に辞書作成作業を行っているが、『風』の辞書も参考にした部分がある。
> よって、『風』のライセンスを持っていない方はご遠慮いただきたい。そういっ
> た特殊な条件による同人的配布で、ダウンロード数を私が確認できるサイトに
> 置いてある。

しかし、2026年現在、考えを変えました。

ダウンロード数が確認できないところでも配布を認めることにしました。動態
保存のためにやむなく、単漢字辞書が必要な場合に使えるようにこの辞書の
「著者」である私がそういう置き方を認めることにしました。[IME「風」](https://www.vector.co.jp/soft/winnt/writing/se122541.html)の更新が長らく止まっている以上、単漢字変換がどこかで使えることのほうが
大事かと思ったのです。私自身は「風」を持ってない方が単漢字変換を使いた
いというのを支援したいので、そういう方がダウンロードするのも許可したい
です。上の条件は後からでも「風」がシェアウェアとしてまだ正式に買えるこ
とを前提としていましたが、今ではそれも明らかではありません。ただ、この
ような条件に問題がある場合は私に何らかの方法でご連絡ください。

あと、もし、「風」が今買えるなら、または将来再び買えるようになったら、
「風」を使わなくても、この辞書を(「おためし」を超えて)使うなら「風」は
買っていただきたいです。

私自身に著作権があるとすれば、できればパブリックドメインであることを望
みますが、それだと使いにくいという場合は、MIT License で…といったとこ
ろです。
